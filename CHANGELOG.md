# Changelog

All notable changes to the loredex Obsidian plugin are documented here. Format
follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/); versioning is
[SemVer](https://semver.org/). Releases (with `manifest.json` + `main.js`) are on
the [releases page](https://github.com/ahmedtawfeeq1/loredex-obsidian/releases).

## [0.3.0] - 2026-07-12

### Added
- **By-product Dashboard view** — the Bases dashboard gains a product-scoped
  lens, aligned with the loredex 2.4.0 core's product scoping.

## [0.2.0] - 2026-07-12

### Changed
- Dashboard now opens the **native Obsidian Bases view** instead of a custom
  markdown view.
- Requires the loredex 2.x core (`Dashboard.base` needs 2.x).

## [0.1.0] - 2026-07-05

### Added
- Initial plugin — Dashboard view, vault sync, and an in-app MCP server exposing
  the loredex vault to MCP-compatible agents (adds `active_note` over the CLI's
  tool set).
